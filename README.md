## 🛠 Tecnologías Utilizadas  
- **Lenguaje**: Kotlin (Android Nativo)  
- **Arquitectura**: MVVM  
- **Librerías**:  
  - Jetpack Compose (UI)  

## 🚀 Funcionalidades Clave  
✅ **UI/UX fiel al diseño original** de Lidl.  
✅ **Validaciones en tiempo real** (email, contraseña).
